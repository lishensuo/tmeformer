TMEformer Documentation
=======================

**TMEformer** is a Tumor Microenvironment (TME) extension of Geneformer, designed for modeling TME information in single-cell spatial transcriptomics data.

.. note::
   This documentation covers the TME-specific modules added to TMEformer, including the :doc:`tme` and :doc:`cli` modules.
